# Chappell Family Tree

A flagship genealogy and family-history application in the ChapNetAI ecosystem.

## Product identity

- Parent company: ChapNetAI
- Category: History
- Application: Chappell Family Tree
- Creator: Brandon D. Chappell
- Repository: bigchap13/chappell-family-tree

## Mission

Chappell Family Tree will preserve, organize, research, and present the history of the Chappell family through an interactive family tree, individual profiles, relationships, historical records, photographs, documents, maps, timelines, stories, and public-record research.

## Planned capabilities

- Interactive ancestor and descendant trees
- Person profiles
- Parent, child, spouse, sibling, adoption, and guardian relationships
- Birth, death, marriage, divorce, burial, residence, education, occupation, and military records
- Photos, documents, obituaries, certificates, letters, videos, and audio memories
- Family timelines
- Migration and location maps
- Public-record research workspace
- Source citations and evidence tracking
- Duplicate-person detection
- Relationship calculator
- GEDCOM import and export
- Living-person privacy controls
- AI genealogy companion
- Printable family books and reports

## Development rules

- Phone-first interface
- Centered flagship header and navigation
- Large readable text
- Premium ChapNetAI visual quality
- No invented family facts
- Every historical claim must support sources and confidence status
- Living-person information must remain private by default
- Public-record searches must respect source terms, copyright, access controls, and privacy laws

## Initial phase

Repository foundation, project canon, application structure, genealogy data model, public-record source registry, and flagship dashboard.
